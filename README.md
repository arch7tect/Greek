# Greek

Персональная вики по изучению новогреческого языка. Исходные PDF, учебники,
конспекты и домашние задания складываются в `inbox/`; обработанные знания
публикуются как связанные Markdown-страницы в `docs/`.

## Быстрый старт

Требования: Python 3.13+ и [uv](https://docs.astral.sh/uv/).

```bash
uv sync
uv run mkdocs serve
```

После запуска вики доступна по адресу <http://127.0.0.1:8000>.

Проверить сборку без запуска сервера:

```bash
uv run mkdocs build --strict
```

## Как добавлять материалы

1. Положить исходный файл в тематическую подпапку `inbox/`, не изменяя оригинал.
2. Сообщить, что это за материал и к какому уроку он относится, если это известно.
3. После обработки проверить созданные страницы, примеры и ответы.
4. Переместить разобранный оригинал в `sources/files/` и зафиксировать его карточку
   в `docs/sources/`.

Подробный процесс и правила именования описаны в
[`CONTRIBUTING.md`](CONTRIBUTING.md).
