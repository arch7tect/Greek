# Извлечённые данные

Этот каталог содержит производные машинные извлечения из оригиналов в
`materials/`. Они нужны как поисковый и проверочный слой, но не являются
неизменяемыми оригиналами и не публикуются в MkDocs автоматически.

- `books/` — постраничный LLM-OCR постоянных учебников.

Каждый OCR-файл указывает исходный PDF, модель, диапазон страниц и качество.
Машинный текст остаётся черновиком до визуальной сверки; `[проверить]`
обозначает неуверенно прочитанный фрагмент.
