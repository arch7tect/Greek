# Мой греческий

Это персональная база знаний по новогреческому языку: история занятий,
справочники, словарь, домашние задания, тренажёры и материалы для повторения.

!!! tip "Куда идти сейчас"
    Продолжи с [маршрута обучения](learning-path.md): там указаны текущая тема
    и ближайшие шаги.
