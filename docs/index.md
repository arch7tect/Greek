# Мой греческий

Это персональная база знаний по новогреческому языку: история занятий,
справочники, словарь, упражнения, домашние задания и материалы для повторения.

!!! tip "Куда идти сейчас"
    Начни с [маршрута обучения](learning-path.md) или открой последний урок.

## Разделы

- [Уроки](lessons/index.md) — что и когда проходили.
- [Справочник](reference/index.md) — грамматика, чтение и устойчивые правила.
- [Словарь](vocabulary/index.md) — слова по темам и урокам.
- [Коротко для памяти](memory/index.md) — компактные памятки.
- [Упражнения](exercises/index.md) — практика и разбор ошибок.
- [Домашние задания](homework/index.md) — задания, решения и обратная связь.
- [Тренажёры](training/index.md) — активное вспоминание и повторение.
- [Источники](sources/index.md) — откуда взялись знания.
