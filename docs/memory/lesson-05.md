# Урок 05 — коротко для памяти

## Окончания первого спряжения

`μέν-ω, μέν-εις, μέν-ει, μέν-ουμε, μέν-ετε, μέν-ουν(ε)`
`[ˈmeno, ˈmenis, ˈmeni, ˈmenume, ˈmenete, ˈmenun(e)]`

Отрицание — `δεν` `[ðen]` перед глаголом: `δεν ξέρω` `[ðen ˈksero]`.

Третье лицо единственного числа одно для всех родов:
`αυτός / αυτή / αυτό μένει` `[afˈtos / afˈti / afˈto ˈmeni]`.

## Как дела — варианты ответа

- Хорошо: `Τέλεια` `[ˈtelia]`, `Μια χαρά` `[mya haˈra]`,
  `Πολύ καλά` `[poˈli kaˈla]`, `Καλά` `[kaˈla]`.
- Сдержанно: `Καλούτσικα` `[kaˈlutsika]`, `Έτσι κι έτσι` `[ˈetsi ki ˈetsi]`,
  `Όχι και τόσο καλά` `[ˈohi ke ˈtoso kaˈla]`.
- Плохо: `Χάλια` `[ˈhalya]`.

## Когда: вопрос и союз

- `Πότε διαβάζετε;` `[ˈpote ðiaˈvazete?]` — «Когда вы занимаетесь?» (вопрос);
- `όταν είσαι κοντά μου` `[ˈotan ˈise koˈnda mu]` — «когда ты рядом» (союз).
