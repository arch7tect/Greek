# Урок 06 — коротко для памяти

## Десятки

`είκοσι, τριάντα, σαράντα, πενήντα, εξήντα`
`[ˈikosi, triˈanda, saˈranda, peˈninda, eˈksinda]`

`εβδομήντα, ογδόντα, ενενήντα, εκατό`
`[evðoˈminda, oɣˈðonda, eneˈninda, ekaˈto]`

Составные — сложением: `σαράντα δύο` `[saˈranda ˈðio]` — 42.

## Телефонные группы

- надёжный вариант — по цифрам: `210` — `δύο, ένα, μηδέν`
  `[ˈðio, ˈena, miˈðen]`;
- можно группировать для тренировки: `210 → 2 | 10` — `δύο, δέκα`
  `[ˈðio, ˈðeka]`, `2586 → 25 | 86` — `είκοσι πέντε, ογδόντα έξι`
  `[ˈikosi ˈpende, oɣˈðonda ˈeksi]`;
- ноль никогда не пропускаем.

Письменный блок из трёх цифр не обязан читаться как число с сотнями.

## Четыре предлога с винительным

- `από` `[aˈpo]` — из, от; `σε` `[se]` — в, к (сливается: `στον/στην/στη/στο`);
- `με` `[me]` — с (и инструмент: `με το μαχαίρι` `[me to maˈheri]` — ножом);
- `για` `[ya]` — для; о (`Μιλάμε για τη δουλειά` `[miˈlame ya ti ðuˈlya]`).

## В секретариате

- `Πώς λέγεστε;` `[pos ˈleyeste?]` — как вас зовут? (вежливо)
- `Μάλιστα` `[ˈmalista]` — да, конечно.
- `Ορίστε` `[oˈriste]` — вот, пожалуйста.
- `μέιλ / παπάκι / τελεία` `[ˈmeil / paˈpaki / teˈlia]` — имейл / @ / точка.

Ударение решает: `η τελεία` `[i teˈlia]` — точка, `τέλεια` `[ˈtelia]` — отлично.
