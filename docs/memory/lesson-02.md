# Урок 02 — коротко для памяти

## `Είμαι` `[ˈime]`

`Εγώ είμαι` `[eˈɣo ˈime]` · `εσύ είσαι` `[eˈsi ˈise]` ·
`αυτός / αυτή / αυτό είναι` `[afˈtos / afˈti / afˈto ˈine]`

`Εμείς είμαστε` `[eˈmis ˈimaste]` · `εσείς είστε` `[eˈsis ˈiste]` ·
`αυτοί / αυτές / αυτά είναι` `[afˈti / afˈtes / afˈta ˈine]`

## Представиться

- `Με λένε...` `[me ˈlene...]` — меня зовут...
- `Είμαι ο...` `[ˈime o...]` — я... (мужчина).
- `Είμαι η...` `[ˈime i...]` — я... (женщина).

## Спросить, откуда человек

- `Από πού είσαι;` `[aˈpo pu ˈise]` — один человек на «ты».
- `Από πού είστε;` `[aˈpo pu ˈiste]` — один человек на «вы» или группа.
- `Είμαι από την Κύπρο` `[ˈime aˈpo tin ˈkipro]` — я с Кипра.

## При знакомстве и прощании

- `Χαίρω πολύ` `[ˈhero poˈli]` — очень приятно при знакомстве.
- `Χάρηκα πολύ` `[ˈharika poˈli]` — было очень приятно при завершении беседы.
- `Κι εγώ` `[ki eˈɣo]` — мне тоже.
- `Τα λέμε` `[ta ˈleme]` — увидимся.
