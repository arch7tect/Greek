# Урок 07 — коротко для памяти

## Артикли в именительном

| Род | Известный предмет | Один новый предмет |
|---|---|---|
| мужской | `ο` `[o]` | `ένας` `[ˈenas]` |
| женский | `η` `[i]` | `μία` `[ˈmia]` |
| средний | `το` `[to]` | `ένα` `[ˈena]` |

Сначала запоминаем род слова, затем выбираем известность:
`μία βιβλιοθήκη → η βιβλιοθήκη`
`[ˈmia vivlioˈθiki → i vivlioˈθiki]`.

## Об учёбе

- `Τι σπουδάζεις;` `[ti spuˈðazis?]` — что ты изучаешь?
- `Σπουδάζω Νομικά.` `[spuˈðazo nomiˈka]` — изучаю право.
- `Κάνω μεταπτυχιακό.` `[ˈkano metaptihiaˈko]` — учусь в магистратуре.
- `Τι γλώσσες μιλάς;` `[ti ˈɣloses miˈlas?]` — на каких языках говоришь?

## Три разные формы

`η Γαλλία → η Γαλλίδα → τα γαλλικά`
`[i ɣaˈlia → i ɣaˈliða → ta ɣaliˈka]`

страна → женщина этой национальности → язык.

## После предлога

`από, σε, με, για` `[aˈpo, se, me, ya]` требуют винительного:
`από τον πίνακα` `[aˈpo ton ˈpinaka]`,
`με την κάρτα` `[me tin ˈkarta]`,
`στο εστιατόριο` `[sto estiaˈtorio]`.
