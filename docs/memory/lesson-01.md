# Урок 01 — быстрое повторение

## Пять гласных звуков

`α [a]` · `ε/αι [e]` · `η/ι/υ/ει/οι/υι [i]` · `ο/ω [o]` · `ου [u]`

`αυ`: `[av]` перед звонким, `[af]` перед глухим.<br>
`ευ`: `[ev]` перед звонким, `[ef]` перед глухим.

## Согласные сочетания

`μπ [b/mb]` · `ντ [d/nd]` · `γκ/γγ [g/ŋg]` · `τσ [ts]` · `τζ [dz]`

## Ударение

- Многосложное слово: ищи тонос — `μαμά` `[maˈma]`.
- Односложное: обычно без тоноса — `με` `[me]`, `σε` `[se]`.
- Важные исключения: `πώς` `[pos]`, `πού` `[pu]`, `ή` `[i]`.
- `αί` `[e]` читается вместе, `άι` `[ai]` — раздельно.

## Мини-диалог

> — Γεια σου! Πώς σε λένε; `[ya su! pos se ˈlene?]`<br>
> — Με λένε ... Εσένα; `[me ˈlene ... eˈsena?]`<br>
> — Με λένε ... Τα λέμε! `[me ˈlene ... ta ˈleme!]`<br>
> — Γεια! `[ya!]`
