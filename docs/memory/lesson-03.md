# Урок 03 — коротко для памяти

## Местоимения + `είμαι` `[ˈime]`

`Εγώ είμαι` `[eˈɣo ˈime]` · `εσύ είσαι` `[eˈsi ˈise]` ·
`αυτός / αυτή / αυτό είναι` `[afˈtos / afˈti / afˈto ˈine]`

`Εμείς είμαστε` `[eˈmis ˈimaste]` · `εσείς είστε` `[eˈsis ˈiste]` ·
`αυτοί / αυτές / αυτά είναι` `[afˈti / afˈtes / afˈta ˈine]`

## Один человек или группа

- `Καλώς ήρθες!` `[kaˈlos ˈirθes]` — одному человеку на «ты».
- `Καλώς ήρθατε!` `[kaˈlos ˈirθate]` — одному человеку на «вы» или группе.
- `Εσάς, πώς σας λένε;` `[eˈsas, pos sas ˈlene?]` — а вас как зовут?

## Говорим о человеке или обращаемся

- `ο κύριος Πέτρος` `[o ˈkirios ˈpetros]` — господин Пётр.
- `Γεια σας, κύριε Πέτρο!` `[ya sas, ˈkirie ˈpetro]` — здравствуйте,
  господин Пётр!

## Принадлежность

- `το βιβλίο μου` `[to viˈvlio mu]` — моя книга;
- `ο καθηγητής σας` `[o kaθiyiˈtis sas]` — ваш преподаватель.
