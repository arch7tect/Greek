# Урок 04 — коротко для памяти

## Два вопроса

- `Από πού είσαι;` `[aˈpo pu ˈise?]` — откуда ты родом?
- `Πού μένεις;` `[pu ˈmenis?]` — где ты живёшь?

## После предлогов

`Από` `[aˈpo]`: `τον / την / το` `[ton / tin / to]`

`Σε` `[se]`: `στον / στην / στο` `[ston / stin / sto]`

> Είμαι από τον Καναδά. `[ˈime aˈpo ton kanaˈða.]`<br>
> Μένω στην Καισαριανή. `[ˈmeno stin kesariaˈni.]`

## Как дела

- `Τι κάνεις;` `[ti ˈkanis?]` — одному на «ты».
- `Τι κάνετε;` `[ti ˈkanete?]` — одному на «вы» или группе.
- `Πολύ καλά, ευχαριστώ` `[poˈli kaˈla, efhariˈsto]` — очень хорошо, спасибо.

## Числа

`έξι, εφτά, οχτώ, εννιά, δέκα` `[ˈeksi, eˈfta, oˈhto, eˈnya, ˈðeka]`
