## Сейчас

- Уровень курса: A1, основной учебник охватывает A1–A2.
- Текущая тема: артикли в именительном падеже, учебная среда и разговор
  об учёбе.
- Разобраны [уроки 01–07](lessons/index.md); лексика собрана в

## Прогресс тренировок

Результаты берутся из тренажёров и хранятся только в этом браузере: на другом
устройстве список будет свой. Сводка учитывает фильтр «Материал» из
тренажёров — карточки позже выбранного урока не считаются. «Уверенно» — карточка отвечена верно и не
числится слабой.
