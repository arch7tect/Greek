## Техническое состояние

PDF представляет собой цветной фотографический скан без полезного текстового
слоя. Метаданные задают поворот страниц на 180 градусов; рендерер PDF применяет
его корректно.

В проверенном диапазоне печатная нумерация опережает PDF-нумерацию на десять:
печатная страница 142 находится на PDF-странице 132. При дальнейшей обработке
соответствие нужно проверять для каждого диапазона.
