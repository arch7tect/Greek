# Источники

Каталог учебников, PDF, раздаточных материалов и заданий. Карточка источника
фиксирует оригинальный файл, диапазон обработанных страниц, качество извлечения и
созданные страницы вики.

## Учебники

- [Ταξίδι στην Ελλάδα 1](taxidi-stin-ellada-1.md) — основной учебник, уровни
  A1–A2; ожидает последовательного разбора.
- [Ταξίδι στην Ελλάδα 1+ — Βιβλίο Ασκήσεων](taxidi-stin-ellada-1-workbook.md) —
  рабочая тетрадь, уровни A1–A2.
- [Αυτό ακριβώς! — Επίπεδο Α΄](afto-akrivos-a.md) — грамматические таблицы и
  упражнения начального уровня.
