# Урок 02 — новые слова

**Статус:** проверено по изображениям и пяти аудиозаписям

**Связанный урок:** [Урок 02 — `είμαι` `[ˈime]`, знакомство и происхождение](../lessons/lesson-02-eimai-and-origin.md)

В активный словарь вошли 76 новых единиц: 60 из назначенных упражнений и 16
полезных слов и формул из заданного для прослушивания диалога. Повторявшиеся
`η βιολογία` `[i violoˈyia]` и `η γεωγραφία` `[i yeoɣraˈfia]` уже находятся в
словаре урока 01 и второй раз не считаются.

Обозначения: [как читать транскрипцию](../reference/transcription.md).

## Учебник, страница 25, упражнение 17 { #lesson-02-ex-17 }

| Слово | Транскрипция | Значение | Форма / примечание |
|---|---|---|---|
| `ευχαριστώ` | `[efhariˈsto]` | спасибо; благодарю | глагол / формула вежливости |
| `γεια` | `[ya]` | привет; пока | формула общения |
| `η Αίγυπτος` | `[i ˈeyiptos]` | Египет | название страны |
| `δουλεύω` | `[ðuˈlevo]` | работать | глагол |
