# Словарь

Словарь строится только из лексики, которая действительно встречалась в
упражнениях или была дана в задании.

Практика: [тренажёр слов по урокам](../training/vocabulary.md).

## Два уровня

- [Общий словарь](all.md) — накопительный список по всем пройденным урокам.
- [Урок 01 — новые слова](lesson-01.md) — слова из упражнений 2, 9 и 10.
- [Урок 02 — новые слова](lesson-02.md) — слова из упражнений 11–13 и 17,
  а также выбранная лексика из заданного диалога.
- [Урок 03 — новые слова](lesson-03.md) — слова из упражнений 2–3 рабочей
  тетради и выбранная лексика из заданного диалога.
- [Урок 04 — новые слова](lesson-04.md) — формы из упражнения 2β, диалог
  упражнения 4 и числа из упражнений 7–8.

Словари уроков — единственный ручной источник. Общий словарь и данные
тренажёра генерируются из них. В колонке «Основное» значение «да» отмечает
первый проход тренажёра: базовое общение, текущую грамматику и лексику заданий.
«—» оставляет в полном наборе более редкие слова из фонетических упражнений.

## Правила отбора

- Слово добавляется, если оно встречается в выполненном или заданном упражнении.
- Для существительного сохраняются артикль и множественное число.
- Для глагола сохраняется словарная форма первого лица единственного числа.
- Для каждого слова и его греческих форм указывается упрощённая транскрипция в
  квадратных скобках по [общим правилам](../reference/transcription.md).
- Имена людей выносятся отдельно и не считаются активной лексикой.
- Если слово нельзя однозначно восстановить без аудио, оно ждёт проверки и не
  попадает в общий словарь.
- После появления аудио запись получает отметку о проверке, а неоднозначные
  варианты заменяются фактически произнесённым словом.
- Каждая запись содержит урок и конкретное упражнение, чтобы её можно было
  проверить по источнику.
