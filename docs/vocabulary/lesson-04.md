# Урок 04 — новые слова

**Статус:** проверено по полным страницам учебников и изображениям заданий

**Связанный урок:** [Урок 04 — числа и место жительства](../lessons/lesson-04-numbers-and-residence.md)

В активный словарь вошли 27 новых единиц: 5 из грамматического учебника, 11 из
диалога рабочей тетради и 11 числительных от 0 до 10. Лексика песни и
дополнительных видео не включена, поскольку в задании нет указания её учить.

Обозначения: [как читать транскрипцию](../reference/transcription.md).

## `Αυτό ακριβώς!` `[afˈto akriˈvos]`, страница 142, упражнение 2β { #lesson-04-afto-ex-2b }

| Слово | Транскрипция | Значение | Форма / примечание |
|---|---|---|---|
| `ο Καναδάς` | `[o kanaˈðas]` | Канада | после `από` `[aˈpo]`: `τον Καναδά` `[ton kanaˈða]` |
| `δεν` | `[ðen]` | не | отрицательная частица перед глаголом |
| `τώρα` | `[ˈtora]` | сейчас | наречие |
| `η γραμματεία` | `[i ɣramaˈtia]` | секретариат; администрация | `οι γραμματείες` `[i ɣramaˈties]` |
| `μας` | `[mas]` | наш; наша; наше | ставится после существительного |

`Είμαι` `[ˈime]`, местоимения, `η Βραζιλία` `[i vraziˈlia]`,
`η τάξη` `[i ˈtaksi]`, `η δασκάλα` `[i ðaˈskala]` и `εδώ` `[eˈðo]` уже
находятся в общем словаре. Имена людей не считаются активной лексикой.

## Рабочая тетрадь, страница 11, упражнение 4 { #lesson-04-workbook-ex-4 }

| Слово | Транскрипция | Значение | Форма / примечание |
|---|---|---|---|
| `τι` | `[ti]` | что; как в вопросе о делах | `Τι κάνεις;` `[ti ˈkanis?]` — как дела? |
| `κάνω` | `[ˈkano]` | делать | `κάνεις` `[ˈkanis]`, `κάνετε` `[ˈkanete]` |
| `πολύ` | `[poˈli]` | очень; много | наречие |
| `καλά` | `[kaˈla]` | хорошо | наречие; ответ о самочувствии |
| `η αλήθεια` | `[i aˈliθia]` | правда | `οι αλήθειες` `[i aˈliθies]`; в диалоге «кстати / правда» |
| `πού` | `[pu]` | где; куда | вопросительное наречие |
| `μένω` | `[ˈmeno]` | жить; оставаться | `μένεις` `[ˈmenis]`, `μένουμε` `[ˈmenume]`, `μένετε` `[ˈmenete]` |
| `η Καισαριανή` | `[i kesariaˈni]` | Кесариани | район Афин |
| `το Παγκράτι` | `[to paŋˈgrati]` | Панграти | район Афин |
| `κοντά` | `[koˈnda]` | близко; рядом | наречие |
| `ακριβώς` | `[akriˈvos]` | точно; именно | `Πού ακριβώς;` `[pu akriˈvos?]` — где именно? |

Имена `Σεργκέι` `[serˈɣei]`, `Ιουλία` `[iuˈlia]` и названия улиц
`Λυδίας` `[liˈðias]`, `Νάξου` `[ˈnaksu]` нужны для диалога, но не входят в
активный словарь.
