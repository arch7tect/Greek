# Урок 03 — новые слова

**Статус:** проверено по рабочей тетради, презентации и аудиозаписи

**Связанный урок:** [Урок 03 — местоимения и знакомство в классе](../lessons/lesson-03-pronouns-and-classroom-dialogue.md)

В активный словарь вошла 21 новая единица: 12 из назначенных упражнений и 9 из
заданного для прослушивания диалога. Набор Quizlet отсутствует среди исходников,
поэтому неизвестные слова из него не добавлены.

Обозначения: [как читать транскрипцию](../reference/transcription.md).

## Рабочая тетрадь, страница 10, упражнение 2 { #lesson-03-ex-2 }

| Слово | Транскрипция | Значение | Форма / примечание |
|---|---|---|---|
| `εγώ` | `[eˈɣo]` | я | личное местоимение |
| `εσύ` | `[eˈsi]` | ты | личное местоимение |
| `εμείς` | `[eˈmis]` | мы | личное местоимение |
| `εσείς` | `[eˈsis]` | вы | вежливо одному человеку или нескольким людям |

Формы `αυτός / αυτή / αυτό` `[afˈtos / afˈti / afˈto]` и глагол
`είμαι` `[ˈime]` уже внесены на уроке 02. Упражнение добавляет множественные
формы `αυτοί / αυτές / αυτά` `[afˈti / afˈtes / afˈta]`, но они относятся к той
же словарной статье и не считаются новыми единицами. Имена людей не учитываются.

## Рабочая тетрадь, страница 11, упражнение 3 { #lesson-03-ex-3 }

| Слово | Транскрипция | Значение | Форма / примечание |
|---|---|---|---|
| `η Ιταλία` | `[i itaˈlia]` | Италия | название страны |
| `η Βραζιλία` | `[i vraziˈlia]` | Бразилия | название страны |
| `η Σμύρνη` | `[i ˈzmirni]` | Смирна; Измир | название города |
| `η κυρία` | `[i kiˈria]` | госпожа; вежливое обращение к женщине | `οι κυρίες` `[i kiˈries]` |
| `η δασκάλα` | `[i ðaˈskala]` | учительница | `οι δασκάλες` `[i ðaˈskales]` |
| `η Τουρκία` | `[i turˈkia]` | Турция | название страны |
| `η Βουλγαρία` | `[i vulɣaˈria]` | Болгария | название страны |
| `μου` | `[mu]` | мой; моя; моё | ставится после существительного |

`Το βιβλίο` `[to viˈvlio]`, `το μολύβι` `[to moˈlivi]` и `είμαι` `[ˈime]`
повторяются из предыдущих уроков. Имена людей из предложений не считаются
активной лексикой.

## Диалог для прослушивания { #lesson-03-dialogue }

| Слово или формула | Транскрипция | Значение | Форма / примечание |
|---|---|---|---|
| `εδώ` | `[eˈðo]` | здесь; сюда | наречие |
| `εμένα` | `[eˈmena]` | меня; а меня | ударная форма первого лица |
| `εσάς` | `[eˈsas]` | вас; а вас | ударная вежливая или множественная форма |
| `η Γερμανία` | `[i yermaˈnia]` | Германия | название страны |
| `καλώς ήρθες / καλώς ήρθατε` | `[kaˈlos ˈirθes / kaˈlos ˈirθate]` | добро пожаловать | одному на «ты» / одному на «вы» или группе |
| `τέλεια` | `[ˈtelia]` | отлично; замечательно | разговорная реакция |
| `ο κύριος` | `[o ˈkirios]` | господин; вежливое обращение к мужчине | `οι κύριοι` `[i ˈkirii]`; обращение `κύριε` `[ˈkirie]` |
| `χαίρετε` | `[ˈherete]` | здравствуйте | нейтрально-вежливое приветствие |
| `σας` | `[sas]` | вас; ваш | значение определяется конструкцией |

Уже известные `η τάξη` `[i ˈtaksi]`, `η Ισπανία` `[i ispaˈnia]`,
`ο καθηγητής` `[o kaθiyiˈtis]`, `το παιδί` `[to peˈði]` и речевые формулы
предыдущих уроков повторно не считаются. Имена участников диалога вынесены за
пределы активного словаря.
