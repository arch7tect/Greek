# Тренажёры

- [Буквы и буквосочетания](alphabet-and-combinations.md) — узнавание букв,
  обратное соответствие «звук → буква» и правила чтения сочетаний.

Следующие возможные форматы:

- карточки на активное вспоминание;
- перевод с русского на греческий и обратно;
- выбор правильной формы слова;
- сборка фразы из частей;
- мини-диалоги по ситуации;
- повторение ошибок из уроков и домашних заданий.

Тренажёры строятся только на уже проверенном материале уроков и справочника.
