# Тренажёры

- [Буквы и буквосочетания](alphabet-and-combinations.md) — узнавание букв,
  обратное соответствие «звук → буква» и правила чтения сочетаний.
- [`είμαι` `[ˈime]` и личные местоимения](eimai-and-pronouns.md) — формы глагола,
  обратное соответствие и короткие фразы с пропусками.
- [Слова по урокам](vocabulary.md) — основной и полный наборы,
  направления перевода, артикли и сохранение прогресса.
- [Числа от 0 до 10](numbers.md) — цифра и слово в обе стороны,
  чтение адресов и варианты `εφτά / επτά` `[eˈfta / eˈpta]`.
- [Винительный после `από` `[aˈpo]` и `σε` `[se]`](accusative-after-apo-se.md) —
  выбор `τον / την / τη / το` `[ton / tin / ti / to]` и слитных форм
  `στον / στην / στη / στο` `[ston / stin / sti / sto]` с объяснением правила.
- [Приветствия и знакомство](greetings.md) — реплика по ситуации:
  поздороваться и попрощаться, познакомиться, спросить о делах.

Следующие возможные форматы:

- сборка фразы из частей;
- мини-диалоги по ситуации;
- повторение ошибок из уроков и домашних заданий.

Тренажёры строятся только на уже проверенном материале уроков и справочника.
