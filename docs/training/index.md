# Тренажёры

Будущие форматы практики:

- карточки на активное вспоминание;
- перевод с русского на греческий и обратно;
- выбор правильной формы слова;
- сборка фразы из частей;
- мини-диалоги по ситуации;
- повторение ошибок из уроков и домашних заданий.

Конкретные тренажёры появятся, когда накопится первый проверенный материал.
