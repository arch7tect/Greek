# Числа от 0 до 10

**Статус:** проверено по основному учебнику, печатная страница 31

**Связанный урок:** [Урок 04 — числа и место жительства](../lessons/lesson-04-numbers-and-residence.md)

| Цифра | По-гречески | Транскрипция |
|---:|---|---|
| 0 | `μηδέν` | `[miˈðen]` |
| 1 | `ένα` | `[ˈena]` |
| 2 | `δύο` | `[ˈðio]` |
| 3 | `τρία` | `[ˈtria]` |
| 4 | `τέσσερα` | `[ˈtesera]` |
| 5 | `πέντε` | `[ˈpende]` |
| 6 | `έξι` | `[ˈeksi]` |
| 7 | `εφτά / επτά` | `[eˈfta / eˈpta]` |
| 8 | `οχτώ / οκτώ` | `[oˈxto / oˈkto]` |
| 9 | `εννιά / εννέα` | `[eˈnja / eˈnea]` |
| 10 | `δέκα` | `[ˈðeka]` |

Для 7, 8 и 9 учебник даёт по два нормативных варианта. Разговорные
`εφτά / οχτώ / εννιά` `[eˈfta / oˈxto / eˈnja]` особенно частотны в устной
речи; `επτά / οκτώ / εννέα` `[eˈpta / oˈkto / eˈnea]` нейтральны и часто
встречаются в официальной речи.

Не смешиваем части разговорного и нейтрального вариантов внутри одного слова.

## Минимальная практика

- `Λυδίας 10` `[liˈðias ˈðeka]` — улица Лидиас, дом 10;
- `Νάξου 3` `[ˈnaksu ˈtria]` — улица Наксу, дом 3;
- `Σποράδων 10` `[spoˈraðon ˈðeka]` — улица Спорадон, дом 10.
