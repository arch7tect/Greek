# Приветствия и знакомство

**Статус:** проверено по материалам урока 01

**Связанный урок:** [Урок 01 — алфавит и приветствия](../lessons/lesson-01-alphabet-and-greetings.md)

## Поздороваться

| По-гречески | Транскрипция | Значение и употребление |
|---|---|---|
| `Γεια!` | `[ʝa]` | Привет! / Пока! |
| `Γεια σου!` | `[ʝa su]` | Привет! — одному человеку, неформально |
| `Γεια σας!` | `[ʝa sas]` | Здравствуйте! — вежливо или нескольким людям |
| `Καλημέρα!` | `[kaliˈmera]` | Доброе утро! / Добрый день! |
| `Καλησπέρα!` | `[kaliˈspera]` | Добрый вечер! |
| `Γεια χαρά!` | `[ʝa xaˈra]` | Привет! / Пока! — разговорное |
| `Χαίρετε!` | `[ˈçerete]` | Здравствуйте! — нейтрально-вежливое |

## Представиться

| По-гречески | Транскрипция | Перевод |
|---|---|---|
| `Πώς σε λένε;` | `[pos se ˈlene]` | Как тебя зовут? |
| `Πώς σας λένε;` | `[pos sas ˈlene]` | Как вас зовут? |
| `Με λένε ...` | `[me ˈlene ...]` | Меня зовут... |

Пример: `Με λένε Μαρία.` `[me ˈlene maˈria]` — «Меня зовут Мария».
