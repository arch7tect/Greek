# Упрощённая транскрипция новогреческого

Эта система нужна, чтобы прочитать греческое слово до того, как чтение алфавита
станет автоматическим. Она основана на привычной учебной британской
транскрипции из `inbox/other/transcription.md` и дополнена звуками
новогреческого языка.

## Общие правила записи

1. Транскрипция заключается в квадратные скобки: `καλημέρα`
   `[kaliˈmera]`.
2. Знак `ˈ` ставится **перед ударным слогом**: `[kali-ˈme-ra]`.
3. В обычной записи границы слогов не показываются: `[kaliˈmera]`. Дефисы
   используются только в объяснениях.
4. Фраза записывается в одной паре скобок: `Γεια σου!` `[ʝa su]`.
5. Артикль входит в транскрипцию существительного: `η σαλάτα`
   `[i saˈlata]`.
6. Транскрипция показывает **произношение**, а не соответствие буква за буквой.

## Гласные

В новогреческом пять гласных звуков. Долгих и кратких фонем нет, поэтому знак
долготы `:` из британской транскрипции не используется.

| Звук | Как читать | Греческая запись | Пример |
|---|---|---|---|
| `a` | «а» | `α` | `σαλάτα` `[saˈlata]` |
| `e` | «э» | `ε`, `αι` | `και` `[ce]` |
| `i` | «и» | `η`, `ι`, `υ`, `ει`, `οι`, `υι` | `είμαι` `[ˈime]` |
| `o` | «о» | `ο`, `ω` | `σώμα` `[ˈsoma]` |
| `u` | «у» | `ου` | `ούζο` `[ˈuzo]` |

Гласные произносятся чисто и не редуцируются так сильно, как в русском языке.

## Обычные согласные

| Символ | Как читать | Основные буквы |
|---|---|---|
| `p` | «п» | `π` |
| `b` | «б» | начальное `μπ` |
| `t` | «т» | `τ` |
| `d` | «д» | начальное `ντ` |
| `k` | твёрдое «к» | `κ` перед `a/o/u` |
| `g` | «г» | начальное `γκ` |
| `m` | «м» | `μ` |
| `n` | «н» | `ν` |
| `f` | «ф» | `φ` |
| `v` | «в» | `β` |
| `s` | «с» | `σ`, `ς` |
| `z` | «з» | `ζ`; иногда `σ` перед звонким согласным |
| `l` | «л» | `λ` |
| `r` | раскатистое «р» | `ρ` |
| `j` | краткое «й» внутри сочетания | например, `ια` в некоторых словах |

## Особые греческие согласные

Эти символы добавлены к исходной британской системе, потому что без них нельзя
достаточно точно передать новогреческое произношение.

| Символ | Как произносить | Греческая запись | Пример |
|---|---|---|---|
| `θ` | глухой английский *th* | `θ` | `θέλω` `[ˈθelo]` |
| `ð` | звонкий английский *th* | `δ` | `διαβάζω` `[ðiaˈvazo]` |
| `ɣ` | звонкий щелевой звук между «г» и «х» | `γ` перед `a/o/u` | `γράφω` `[ˈɣrafo]` |
| `ʝ` | мягкий вариант `ɣ`, близкий к мягкому «й/г» | `γ` перед `e/i` | `γεωγραφία` `[ʝeoɣraˈfia]` |
| `x` | твёрдый «х» | `χ` перед `a/o/u` | `χαρά` `[xaˈra]` |
| `ç` | мягкий «х» | `χ` перед `e/i` | `χαίρετε` `[ˈçerete]` |
| `c` | мягкий «к» | `κ` перед `e/i` | `κιμωλία` `[cimoˈlia]` |
| `ŋ` | носовой «нг» | внутренние `γκ`, `γγ` | `άγγελος` `[ˈaŋgelos]` |
| `ts` | «ц» | `τσ` | `τσάντα` `[ˈtsanda]` |
| `dz` | «дз» | `τζ` | `τζάμι` `[ˈdzami]` |
| `ks` | «кс» | `ξ` | `τάξη` `[ˈtaksi]` |
| `ps` | «пс» | `ψ` | `ψάρι` `[ˈpsari]` |

## Сочетания гласных

| Сочетание | Чтение | Пример |
|---|---|---|
| `αι` | `[e]` | `και` `[ce]` |
| `ει`, `οι`, `υι` | `[i]` | `φοιτητής` `[fitiˈtis]` |
| `ου` | `[u]` | `ούζο` `[ˈuzo]` |
| `αυ` | `[av]` перед гласной или звонкой согласной | `αύριο` `[ˈavrio]` |
| `αυ` | `[af]` перед глухой согласной | `αυτός` `[afˈtos]` |
| `ευ` | `[ev]` перед гласной или звонкой согласной | `Ευρώπη` `[eˈvropi]` |
| `ευ` | `[ef]` перед глухой согласной | `ευχαριστώ` `[efxariˈsto]` |

Если соседние гласные читаются раздельно, это отражается отдельными звуками:
`τσάι` `[ˈtsai]`, `Μάιος` `[ˈmaios]`.

## Сочетания согласных

| Сочетание | В начале слова | Внутри слова | Пример |
|---|---|---|---|
| `μπ` | `[b]` | `[mb]` или `[b]` | `μπάλα` `[ˈbala]`, `ταμπέλα` `[taˈmbela]` |
| `ντ` | `[d]` | `[nd]` или `[d]` | `ντεκόρ` `[deˈkor]`, `πάντα` `[ˈpanda]` |
| `γκ` | `[g]` | `[ŋg]` или `[g]` | `γκάζι` `[ˈgazi]`, `αγκινάρα` `[aŋgiˈnara]` |
| `γγ` | — | `[ŋg]` | `άγγελος` `[ˈaŋgelos]` |
| `τσ` | `[ts]` | `[ts]` | `τσάντα` `[ˈtsanda]` |
| `τζ` | `[dz]` | `[dz]` или `[ndz]` | `τζάμι` `[ˈdzami]` |

Удвоенная согласная обычно не становится длиннее: `φύλλο` `[ˈfilo]`.

## Ударение

- В многосложном слове `ˈ` ставится перед ударным слогом: `μαθητής`
  `[maθiˈtis]`.
- В односложном слове знак не нужен: `φως` `[fos]`.
- Во фразе отмечается ударение каждого самостоятельного слова: `Καλό βράδυ!`
  `[kaˈlo ˈvraði]`.
- Ударение проверяется по греческому тоносу или по записи преподавателя.

## Что система намеренно упрощает

- Не отмечается долгота гласных.
- Не записываются индивидуальный тембр голоса и очень мелкие позиционные
  изменения.
- Варианты произношения `μπ/ντ/γκ` внутри слова уточняются по аудио, когда оно
  доступно.
- Транскрипция помогает начать читать, но не заменяет прослушивание живой речи.

## Короткие примеры

| Греческий | Транскрипция | Перевод |
|---|---|---|
| `Γεια σου!` | `[ʝa su]` | Привет! |
| `Πώς σε λένε;` | `[pos se ˈlene]` | Как тебя зовут? |
| `Με λένε Μαρία.` | `[me ˈlene maˈria]` | Меня зовут Мария. |
| `παρακαλώ` | `[parakaˈlo]` | пожалуйста |
| `το τηλέφωνο` | `[to tiˈlefono]` | телефон |

## Источник и область применения

Основа системы: `inbox/other/transcription.md`. Греческие дополнения описаны на
этой странице и обязательны для словарей, учебных фраз, примеров, диалогов и
памяток. Библиографические названия и имена авторов можно оставлять без
транскрипции.
