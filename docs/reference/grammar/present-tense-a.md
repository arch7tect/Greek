# Настоящее время: глаголы первого спряжения (συζυγία Α)

**Статус:** проверено по материалам урока 05

**Связанный урок:** [Урок 05 — Τι κάνεις; и глаголы первого спряжения](../../lessons/lesson-05-ti-kaneis-and-verbs-a.md)

Практика: [тренажёр первого спряжения](../../training/present-tense-a.md).

К первому спряжению (`συζυγία Α` `[siziˈyia ˈalfa]`) относятся глаголы с
безударным окончанием `-ω` `[o]` в словарной форме: `μένω` `[ˈmeno]`,
`κάνω` `[ˈkano]`, `έχω` `[ˈeho]`.

Педагогическое пояснение (обобщение сверх слайдов урока): лицо и число
выражаются окончанием, поэтому личное местоимение обычно не нужно и
добавляется для контраста или уточнения.

## Окончания на примере `μένω` `[ˈmeno]`

| Лицо | Форма | Транскрипция | Окончание |
|---|---|---|---|
| `εγώ` `[eˈɣo]` | `μένω` | `[ˈmeno]` | `-ω` `[o]` |
| `εσύ` `[eˈsi]` | `μένεις` | `[ˈmenis]` | `-εις` `[is]` |
| `αυτός / αυτή / αυτό` `[afˈtos / afˈti / afˈto]` | `μένει` | `[ˈmeni]` | `-ει` `[i]` |
| `εμείς` `[eˈmis]` | `μένουμε` | `[ˈmenume]` | `-ουμε` `[ume]` |
| `εσείς` `[eˈsis]` | `μένετε` | `[ˈmenete]` | `-ετε` `[ete]` |
| `αυτοί / αυτές / αυτά` `[afˈti / afˈtes / afˈta]` | `μένουν(ε)` | `[ˈmenun(e)]` | `-ουν(ε)` `[un(e)]` |

В третьем лице множественного числа слайд даёт оба варианта:
`μένουν(ε)` `[ˈmenun(e)]`; форма на `-ουνε` `[une]` встречается и в домашнем
упражнении 14 (`Ξέρουνε ελληνικά;` `[ˈkserune eliniˈka?]`). Пометка о её
разговорной окраске — педагогическое пояснение, на слайдах его нет.

## Отрицание

Отрицание строится частицей `δεν` `[ðen]` перед глаголом:
`δεν μένω` `[ðen ˈmeno]` — «я не живу»,
`Αυτές δεν διαβάζουν ελληνικά.` `[afˈtes ðen ðiaˈvazun eliniˈka]` —
«Они не читают по-гречески».

## Как выбрать форму

1. Найти, кто выполняет действие; в третьем лице единственного числа форма
   одна для всех родов: `αυτός / αυτή / αυτό μένει` `[afˈtos / afˈti / afˈto
   ˈmeni]`.
2. Убрать `-ω` `[o]` словарной формы и добавить окончание лица.
3. Ударение остаётся на том же слоге основы: `μέν-ω → μέν-ουμε`
   `[ˈmeno → ˈmenume]` — так во всех формах таблицы слайда; как общее правило
   это педагогическое пояснение.

Частая ошибка — переносить русскую привычку всегда называть местоимение:
по-гречески `Μένω στην Κυψέλη.` `[ˈmeno stin kiˈpseli]` уже означает
«я живу в Кипсели», а `Εγώ μένω…` `[eˈɣo ˈmeno]` звучит как противопоставление
«а вот я живу…». Вторая частая ошибка — использовать окончание `-ει` `[i]`
только для мужского рода: оно одинаково для `αυτός`, `αυτή` и `αυτό`.
