# упрощённая британская транскрипция
Фактически это та система, которую использовали советские и многие британские учебники (например, Hornby, Oxford 60–90-х годов). Она почти совпадает с IPA, но гораздо проще для запоминания. Ниже — практически полный справочник.


---

# Гласные

## Краткие

| Транскрипция | Как читать                         | Пример         |
| ------------ | ---------------------------------- | -------------- |
| **ɪ**        | между русскими «и» и «ы»           | sit /sɪt/      |
| **e**        | русское «э»                        | pen /pen/      |
| **æ**        | широкий звук между «а» и «э»       | cat /kæt/      |
| **ʌ**        | короткое «а» (не русское!)         | cup /kʌp/      |
| **ɒ**        | короткое открытое «о» (британское) | hot /hɒt/      |
| **ʊ**        | короткое «у»                       | book /bʊk/     |
| **ə**        | нейтральный безударный звук (шва)  | about /əˈbaʊt/ |

---

## Долгие

Двоеточие **:** означает долготу.

| Транскрипция | Как читать                                  | Пример      |
| ------------ | ------------------------------------------- | ----------- |
| **i:**       | долгое «и»                                  | see /si:/   |
| **ɑ:**       | долгое «а»                                  | car /kɑ:/   |
| **ɔ:**       | долгое «о»                                  | law /lɔ:/   |
| **u:**       | долгое «у»                                  | food /fu:d/ |
| **ɜ:**       | долгое «ё/э» с центральным положением языка | bird /bɜ:d/ |

---

# Дифтонги

Это два звука, произносимые одним слогом.

| Транскрипция | Приблизительно          |
| ------------ | ----------------------- |
| **eɪ**       | эй                      |
| **aɪ**       | ай                      |
| **ɔɪ**       | ой                      |
| **aʊ**       | ау                      |
| **əʊ**       | оу (британский вариант) |
| **ɪə**       | иа                      |
| **eə**       | эа                      |
| **ʊə**       | уа                      |

Примеры:

```
day      deɪ
time     taɪm
boy      bɔɪ
house    haʊs
go       ɡəʊ
ear      ɪə
hair     heə
tour     tʊə
```

| Символ | Звук                                |
| ------ | ----------------------------------- |
| p      | п                                   |
| b      | б                                   |
| t      | т                                   |
| d      | д                                   |
| k      | к                                   |
| g      | г                                   |

# Особые согласные

Вот что обычно приходится учить отдельно.

| Символ | Как произносить    |
| ------ | ------------------ |
| θ      | глухой th          |
| ð      | звонкий th         |
| ʃ      | ш                  |
| ʒ      | ж (как в *vision*) |
| tʃ     | ч                  |
| dʒ     | дж                 |

Например

```
think      θɪŋk
this       ðɪs
ship       ʃɪp
vision     ˈvɪʒən
chair      tʃeə
job        dʒɒb
```

---

# Ударение

```
ˈ
```

ставится перед ударным слогом.

Например

```
beˈgin
aˈbout
reˈmember
```

Второстепенное ударение

```
ˌ
```

например

```
ˌunderˈstand
```

---

# Долгота

```
:
```

означает только долготу.

```
ship  ʃɪp
sheep ʃi:p

full  fʊl
fool  fu:l
```

---

# Самые важные различия

Очень важно не путать пары:

| Короткий | Долгий |
| -------- | ------ |
| ɪ        | i:     |
| ʊ        | u:     |
| ɒ        | ɔ:     |
| æ        | ɑ:     |
| ʌ        | ɑ:     |

Например

```
ship   sheep
full   fool
cot    caught
hat    heart
```

---

# Что обычно упрощали в учебниках

Во многих советских изданиях можно встретить такие записи:

| Современный IPA | Учебниковая |
| --------------- | ----------- |
| iː              | i:          |
| uː              | u:          |
| ɔː              | o:          |
| ɑː              | a:          |
| ɜː              | ə:          |
| əʊ              | ou          |
| eɪ              | ei          |
| aɪ              | ai          |
| ɔɪ              | oi          |
| aʊ              | au          |

То есть вместо настоящих IPA-символов использовались привычные сочетания латинских букв. Например:

```
name     /neim/
home     /houm/
boy      /boi/
time     /taim/
```

Именно такой вариант многие помнят со школы.

## Если вы хотите снова свободно читать транскрипцию

Я бы рекомендовал учить именно эту "педагогическую" систему, а затем легко перейти к IPA. Разница между ними небольшая — около 10–15 символов. После этого вы сможете без труда читать транскрипцию практически в любом современном словаре.
