# Входящие материалы

Сюда складываются ещё не обработанные оригиналы: PDF, изображения, документы и
текстовые файлы.

Рекомендуемые подпапки:

- `books/` — учебники и справочники;
- `lessons/` — материалы конкретных уроков;
- `homework/` — задания и выполненные работы;
- `other/` — всё остальное.

Имя файла желательно сохранить исходным. Если оно неинформативно, можно добавить
дату и краткое описание без удаления оригинального названия.
