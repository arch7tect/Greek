# Входящие материалы

Сюда складываются ещё не обработанные оригиналы: PDF, изображения, документы и
текстовые файлы.

Рекомендуемые подпапки:

- `books/` — учебники и справочники;
- `lessons/` — материалы конкретных уроков;
- `homework/` — задания и выполненные работы;
- `other/` — всё остальное.

Материалы занятий именуются по номеру урока: `lesson-01-slides.pdf`,
`lesson-01-exercise-09.mp3`, `lesson-02-homework.txt`. Календарные даты в имена
не добавляются.
