# Входящие материалы

Это очередь ещё не обработанных оригиналов: PDF, изображений, документов,
аудиозаписей и текстовых файлов. После успешного разбора файл перемещается в ту
же тематическую подпапку `materials/`. Поэтому пустая подпапка здесь означает,
что очередь по этому типу материалов разобрана.

Рекомендуемые подпапки:

- `books/` — постоянные учебники, рабочие тетради и справочники; для полного
  разбора используется `$process-greek-book`;
- `lessons/` — материалы конкретных уроков;
- `homework/` — задания и выполненные работы;
- `other/` — всё остальное.

Материалы занятий после нормализации именуются по номеру урока: `lesson-01-slides.pdf`,
`lesson-01-exercise-09.mp3`, `lesson-02-homework.txt`. Календарные даты в имена
не добавляются.
